7.9

早上8点上完琴，今天左手手腕外侧还是疼，甚至比昨天严重。
9点下课回家摆烂打王者，刷b站。
下午2点开始学习hexo,butterfly,美化网站。
7点打羽毛球一个小时。
8点半给浩浩讲课一个小时。
打王者一个小时。
心劲似乎慢慢在恢复。空虚不知道干什么的时候就学习。我想学很多东西，很多很多。

7.10

7点起床和wc跑步
python 143-155
mit 线代 1
晚七点锻炼一小时
给浩浩上课一小时
王者

7.11

8点钢琴
python 156-172
讲课一小时

效率有点低，学一个小时就想玩手机。
想对以后的自己说，焦虑的时候，犹豫的时候，不要停下来，继续做，继续走，继续学习。
我喜欢学习。

7.12

七点起床跑步打羽毛球
python 173-175
web前端 1-22
可惜今天没有练琴，不知道手什么时候才能好
晚上烧烤

学习让我充实，独处让我充实。
得学点数学，我爱数学。

7.13

七点半羽毛球
web前端 23-62，学完了html基础，开始css
练琴一小时，手还没好，好了之后一定狠狠练琴
东升烧烤，有点微醺

7.14

梦
老师问我们以后想要成为什么样的人
我说：如果可以，我想成为科学家
老师问班里同学认为我可以的请站起来
我看见一部分人（宋嘉熙）环顾四周，缓缓站起，又想坐下。
最后班里只有不到一半人站着
老师说：看来你不行啊。
梦醒

6.30起床羽毛球，魏琛像个战神一穿三
参观亭口习仲勋革命遗址
回相公，龙头
和曹翼飞吃饭，要的有一点多
三天没有给浩浩上课了，之后该抓紧了
明天得开始招学生了。

7.15

钢琴，周末没怎么练，要是手没有受伤就好了。开始《瑶族长鼓舞》左右手合。
学pr，但好像没有素材和需求，遂听了一节课后放弃
发补课宣传
给浩浩讲课

7.16

又是梦
梦到吻

学前端 62-94
计算机网络 一课，但前置知识好像欠缺的有点多，很多东西听不明白
给浩浩讲课，但好像没有讲什么实质性的东西。

7.17

两天没有早起锻炼了，今天七点去体育场转了两圈。
前端 94-111，每次一直学一门课厌倦之后换一门课学就会很舒服。
去医院，手相比上周好多了，万幸不是腱鞘炎，医生说没什么事。
明天就要开高二的课了，得准备一下。

学习一门科目好像不能一以贯之，应当改正。
